## Snapshot

## Detailed Notes
- Led development of Techu as a cross-platform competitive card game with online multiplayer and AI-driven gameplay.
- The Unity project shows substantial product scope beyond the short resume bullet: Photon networking, PlayFab data and live-ops features, Firebase notifications, analytics, in-app purchase support, card and board systems, progression, and UI systems.
- The AI side spans multiple approaches. The Unity project includes minimax and Monte Carlo Tree Search code, the CV mentions DQN and MCTS, and the standalone `TechuRL` repo adds a Gymnasium environment with MaskablePPO training for policy experiments.
- A later web adaptation in `TechuOnTheWeb` recreates the game using Next.js, React, TypeScript, Tailwind, Zustand, and browser-native game logic, which helps demonstrate that the game rules were formalized beyond the original Unity client.
