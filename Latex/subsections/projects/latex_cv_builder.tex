\subsection{Latex CV Builder}{Personal Project, \href{https://github.com/Danial-Kord/Latex-CV-Builder}{project link}}{2022 - now}
\begin{itemize}
    \item Developed a full-stack resume generator that converts JSON data to PDF using \textbf{Java + Spring Boot} microservices and \textbf{Django}, containerized with Docker and deployed on \textbf{AWS ECS}.
    \item Integrated \textbf{LangChain} and \textbf{n8n} for automated LLM-based resume customization workflows, and deployed a \textbf{React/Next.js} frontend via Vercel with monitoring and error handling.
\end{itemize}
